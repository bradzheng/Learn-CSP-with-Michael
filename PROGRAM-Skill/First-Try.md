# 完整解题过程（Java 实现 · 严格遵循 SOP）
## 设计的第一部分：分析
### 1. 理解问题（用自己的话描述）
**核心目标**：编写 Java 程序，根据选手 A、B 的比赛得分与出手顺序，按照规则判定并输出最终获胜者。
**判定规则优先级**：
① 两位选手得分不同时，得分更高的选手直接获胜；
② 两位选手得分完全相同时，后出手的选手获胜。

### 2. 拆解问题
#### 子任务划分（包含关系）
| 子任务编号 | 子任务描述                          | 核心动作                     |
|------------|-----------------------------------|------------------------------|
| 子任务 1    | 声明并存储选手 A、B 的比赛得分            | 变量定义、赋值初始化           |
| 子任务 2    | 声明并存储「后出手选手」的标识        | 变量定义、赋值初始化           |
| 子任务 3    | 按比赛规则分情况比较，判定最终获胜选手      | 条件分支判断（得分高低 / 同分处理） |
| 子任务 4    | 向控制台输出判定结果与关键信息                    | 打印输出结果                |

#### 依赖关系
- 子任务 3 依赖 子任务 1（得分数据） + 子任务 2（出手顺序数据），是判定逻辑的输入前提；
- 子任务 4 依赖 子任务 3（胜负判定结果），是逻辑的输出环节；
- 子任务 1、2 互相独立，无前置依赖，可按顺序执行。

| 类别       | 关键词                          |
|------------|---------------------------------|
| 目标相关   | 胜负判定、得分比较、得分相等、后出手、获胜者 |
| 组件相关   | 选手A、选手B、得分值、出手顺序、后出手选手 |
| 依赖相关   | 得分高低 → 胜负结果；得分相等 → 出手顺序决定胜负 |

### 4. 定义变量（Java 语言 · 遵循小驼峰命名规范）
| 变量名       | 数据类型 | 含义                 | 取值示例          |
|--------------|----------|----------------------|-------------------|
| `aScore`      | `int`      | 选手 A 的比赛得分          | 90、85、100       |
| `bScore`      | `int`      | 选手 B 的比赛得分          | 88、85、95        |
| `lastPlayer`  | `char`     | 后出手的选手标识         | `'A'`、`'B'`          |
| `winner`      | `char`     | 最终判定的获胜选手标识       | `'A'`、`'B'`          |

## 设计的第二部分：Implementation（Java 实现）
### 1. 定结构 - 控制结构选择
- **顺序结构**：变量声明与初始化、结果打印输出，自上而下依次执行；
- **分支结构**：核心逻辑使用 `if - else if - else` 三级分支，对应三种判定场景，是本题的核心控制结构；
- **循环结构**：本题为单次胜负判定，无重复执行的同性质任务，无需使用循环。

### 2. 完整代码实现
```java
public class GameWinnerJudge {
    public static void main(String[] args) {
        // ========== 步骤1：顺序结构 - 初始化基础变量 ==========
        // 可修改以下变量值，测试不同场景
        int aScore = 85;       // 选手A得分
        int bScore = 85;       // 选手B得分
        char lastPlayer = 'B'; // 后出手的选手：'A' 或 'B'
        char winner;           // 存储最终获胜者

        // ========== 步骤2：分支结构 - 按规则判定胜负 ==========
        if (aScore > bScore) {
            // 场景1：选手A得分更高，A获胜
            winner = 'A';
        } else if (bScore > aScore) {
            // 场景2：选手B得分更高，B获胜
            winner = 'B';
        } else {
            // 场景3：得分相同，后出手的选手获胜
            winner = lastPlayer;
        }

        // ========== 步骤3：顺序结构 - 输出判定结果 ==========
        System.out.println("选手A得分：" + aScore + " | 选手B得分：" + bScore + " | 后出手选手：" + lastPlayer);
        System.out.println("最终获胜者：选手" + winner);
    }
}
```
