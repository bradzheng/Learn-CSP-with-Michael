# 解题分析：16L 水平均分给 4 人

## 一、设计的第一部分：分析

### 1. 理解问题

**Goal：** 有 3 个桶（8L、8L、3L），初始两个 8L 桶装满水共 16L，3L 桶为空。将 16L 水平均分给 4 人（每人 4L）。4 人有容量 >4L 的接水容器，给出的水不能收回。需实现**分水的操作步骤**。

**核心约束：**
- 只能操作 3 个桶
- 给人水是**不可逆操作**
- 每人恰好 4L

---

### 2. 拆解问题

#### 包含关系

| 建模状态 | 3 个桶水量 + 4 人接水量 |

```
初始 [8,8,0] → 桶间倒水(用3L桶量) → 给人(不可逆) → 重复 → 每人4L
```

## 三、数学推理 — 核心策略

> 3L 桶是量具。利用接水杯的**剩余空间**间接量取非 3 的倍数的水量：
> - 人已有3L → 倒1L满 → 桶剩2L
> - 人已有2L → 倒2L满 → 桶剩1L
> - 人已有1L → 倒3L满 → 桶倒空

### 完整方案

```
初始: 桶=[8,8,0], 人=[0,0,0,0]

人0 得4L:
  桶0→桶2(满): [5,8,3]  → 桶2(3)→人0: 人0=3
  桶0→桶2(满): [2,8,3]  → 桶2→人0(需1): 人0=4✓, [2,8,2]

人1 得4L:
  桶2(2)→人1: 人1=2
  桶1→桶2(满): [2,5,3]  → 桶2→人1(需2): 人1=4✓, [2,5,1]

人2 得4L:
  桶2(1)→人2: 人2=1
  桶1→桶2(满): [2,2,3]  → 桶2(3)→人2(需3): 人2=4✓, [2,2,0]

人3 得4L:
  桶0(2)→人3: 人3=2
  桶1(2)→人3(需2): 人3=4✓, [0,0,0]

最终: 桶=[0,0,0], 人=[4,4,4,4] ✓
```
