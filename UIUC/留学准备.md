## 留学准备

### F1 visa — 广州美领馆面签 Checklist（2026/06/12）

---

#### 🔴 必须携带（缺一不可，逐项打勾）

##### 1. 护照与身份
- [ ] **有效护照**（有效期需覆盖在美学习期间，建议至少6个月以上）
- [ ] **DS-160 确认页**（带条形码的那张，打印件）
- [ ] **面签预约确认信**（Appointment Confirmation，打印件）
- [ ] **I-20 表格**（已签字！学生签名栏 + 日期，未满18岁需家长也签）
- [ ] **SEVIS I-901 缴费收据**（打印件）
- [ ] **签证费（MRV）缴费收据**（打印件）
- [ ] **签证照片**（2张，51mm×51mm白底，近6个月内，背后用铅笔写名字）

##### 2. 学校相关（需由高中出具 / 大学出具）
- [ ] **UIUC 录取通知书**（Admission Letter，打印件）
- [ ] **高中毕业证书**（原件 + 英文翻译件，如尚未拿到则带预毕业证明）
- [ ] **高中成绩单**（原件 + 英文翻译件，密封信封更佳）
- [ ] **标化成绩**：托福/雅思/多邻国成绩单（原件或打印件）

##### 3. 财力证明（需由家长提供）
- [ ] **银行存款证明**（中英文，金额覆盖 I-20 上的一年总费用，建议冻结到签证后）
- [ ] **父母收入证明**（中英文，加盖单位公章）
- [ ] **父母近6个月银行流水**
- [ ] **房产证**（原件，辅助证明家庭资产）
- [ ] **户口本**（原件，证明亲属关系）
- [ ] **出生证明 / 亲属关系公证书**（如户口本不能直观体现父母关系，建议带）
- [ ] **父母支持信 / 资助声明**（Affidavit of Support，中英文，父母签字）

##### 4. 学生自身材料
- [ ] **个人简历 / CV**（英文，简明扼要）
- [ ] **学习计划**（Study Plan，英文，说明为什么选UIUC/GGIS专业、学习目标、毕业规划）
- [ ] **毕业后的归国计划**（简要说明，为什么一定会回中国——F1面签核心）
- [ ] **与UIUC的往来邮件**（如有，打印关键几封）

##### 5. 支付收据（再次确认）
- [ ] SEVIS I-901 缴费成功页
- [ ] MRV签证费缴费成功页

---

#### 🟡 建议携带（增加通过率的辅助材料）
- [ ] **全家福照片**（1-2张，展示家庭联系/国内羁绊）
- [ ] **身份证**（原件）
- [ ] **驾照**（如有，原件）
- [ ] **过往出境记录**（旧护照上的签证页/出入境章，如有）

---

#### 🛑 面签当日注意事项
- [ ] 打印地图，确认**广州美领馆地址**：广州市天河区珠江新城华夏路43号（近珠江新城地铁站B1出口）
- [ ] **提前到达**：建议比预约时间早30分钟到达（不要过早，保安按时间段放人）
- [ ] **不能携带的物品**：手机、电子手表、U盘、背包、打火机、食物饮料。领馆不提供寄存，需在对面/附近付费寄存（带好现金零钱）
- [ ] **只带透明文件袋**：所有材料放入一个透明文件袋/风琴包
- [ ] **着装**：整洁得体，无需正装但不要过于随意（smart casual即可）
- [ ] **心态准备**：诚实回答、简洁明了、不要背书、不要紧张
- [ ] **常见问题准备**：
  1. Why this university? → 说明对UIUC GGIS专业的了解和兴趣
  2. Why this major? → 结合自己的兴趣和未来规划
  3. Who will pay for your study? → 父母资助，简洁说明父母职业
  4. What's your plan after graduation? → **一定说回国**，结合国内GIS行业发展机会
  5. Do you have relatives in the US? → 如实回答

---

### 行前准备
#### 体检、疫苗及相关文件
- [ ] 完成学校要求的体检和疫苗接种（UIUC会在录取后发送具体要求）
- [ ] 《国际旅行健康检查证明书》（小红本）
- [ ] 《疫苗接种或预防措施国际证书》（小黄本）
- [ ] UIUC Immunization Form（由医生填写并签字）

#### 携带物品清单（初步，后续细化）
- [ ] 护照 + I-20（随身携带，勿托运）
- [ ] 少量美元现金
- [ ] 常用药品（感冒药、退烧药、肠胃药等，带英文说明）
- [ ] 转换插头（美标）
- [ ] 眼镜（多配一副备用）
- [ ] 笔记本电脑 + 充电器
- [ ] 重要文件复印件（另存一份给家长）
