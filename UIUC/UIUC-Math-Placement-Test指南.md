# UIUC 数学 Placement Test (ALEKS PPL) 完整指南

> 适用对象：UIUC 本科大一新生，入学前必须完成数学 Placement Test
>
> 更新日期：2025 年 6 月（基于 2024-2026 学年官方信息）
>
> 对 CS+GGIS 转专业至关重要：测试成绩直接影响你是否能被放在 MATH 220/221（微积分 I），这是 CS 核心序列的前提。

---

## 一、什么是 ALEKS PPL？

UIUC 采用的数学 Placement Test 是 **ALEKS PPL**（Assessment and LEarning in Knowledge Spaces — Placement, Preparation, and Learning）。它是一个 **自适应、开放作答**（非选择题）的在线数学评估系统。

| 项目 | 详情 |
|------|------|
| 考试名称 | ALEKS PPL Mathematics Assessment |
| 平台 | ALEKS（McGraw-Hill） |
| 题型 | **全部开放作答**（非选择题），约 30 题 |
| 时长 | 平均 **90 分钟**，最长建议预留 **3 小时** |
| 限制 | 必须一次完成，从开始起 **24 小时内**提交 |
| 计算器 | ALEKS **内置提供**（需要时才会出现），不允许使用自己的计算器 |
| 草稿 | 允许使用草稿纸和笔 |
| 费用 | 对新生 **完全免费** |
| 主办方 | UIUC Department of Mathematics + CITL |

---

## 二、谁必须参加？

- ✅ **所有本科大一新生**（无论专业、无论是否有 AP/IB 学分）— 必须参加，无例外
- ✅ **转学生**（如计划选修微积分 I 或更低级别数学课）
- ✅ 即使你已经拿到 AP Calculus 5 分，也必须参加（作为 placement 参考）

## 三、考试范围（Topics Covered）

ALEKS PPL 覆盖从**基础算术到预备微积分**的全部内容。由于是自适应测试，每个人看到的题目不完全相同，但都来自以下范围：

### 1. 算术与实数（Arithmetic & Real Numbers）

- 分数、整数、小数运算
- 百分比、比例、比率
- 运算顺序
- 实数性质
- 估算与数感

### 2. 初级代数（Algebra I）

- 正负数与绝对值
- 一元一次方程与不等式
- 直线方程与斜率截距形式
- 二元一次方程组（消元法、代入法）
- 线性关系应用题

### 3. 中级代数（Algebra II）

- 关系与函数（定义域、值域、函数记号）
- 线性函数及其图像
- 二次方程（因式分解、求根公式、配方法）
- 二次函数（顶点、对称轴、抛物线图像）
- 多项式运算（加、减、乘、除）
- 因式分解（提取公因式、三项式、平方差、分组分解）
- 多项式方程

### 4. 高级代数（College Algebra / Precalculus）

- **有理表达式** — 化简、乘除、有理方程与函数
- **根式** — 化简根式、高次根、分数指数（有理指数）
- **指数与对数** — 指数法则、对数性质、指数方程、对数方程
- **函数复合** — (f ∘ g)(x)、反函数、一一函数

### 5. 几何（Geometry）

- 周长、面积、体积（二维与三维图形）
- 坐标几何（距离公式、中点公式、斜率）
- 勾股定理
- 三角形、圆、多边形的性质

### 6. 三角学（Trigonometry）

- 角度（度数与弧度、终边相同角、参考角）
- 直角三角形三角（SOH CAH TOA）
- 单位圆
- 三角函数（sin, cos, tan 及其倒数）
- 三角函数的图像（振幅、周期、相位移动）
- 反三角函数
- 三角恒等式（Pythagorean、和差公式、二倍角）
- 三角方程
- 正弦定理与余弦定理

## 四、UIUC 分数与课程 Placement 对应表

### 2024-2025 学年 Cutoff（2025-2026 尚未公布，预计类似）

| ALEKS 分数 | 可注册课程 | 说明 |
|:----------:|------------|------|
| **0-19** | MATH 101 / 重新考 | 所有学院（工程除外）；工程学院学生必须学模块后重考 |
| **20-39** | MATH 101 / 准备后 MATH 112 | 需额外准备才能进入更高课程 |
| **40-64** | MATH 112（商业预微积分/代数）| 修完后可进入微积分序列 |
| **65-79** | MATH 115（预微积分）/ MATH 234 / MATH 124 | 可进入微积分预备阶段 |
| **80-100** | **MATH 220（微积分 I）或 MATH 221（微积分 I）** | ⭐ CS+GGIS 目标区间 |

> 📌 **对 CS+GGIS 的关键意义**：
> - MATH 220/221 是 CS 核心课的 co-requisite / 前置
> - 必须拿到 **80+** 才能被放在微积分 I
> - MATH 221 要求有 **1 年高中微积分基础**；MATH 220 面向少或无微积分背景
> - MATH 231（微积分 II）也是 CS+GGIS 必修，placement 直接影响进度

### 各学院差异

不同学院（LAS、Engineering、Business 等）的课程推荐路径略有不同。务必查阅 [CITL Math Placement 页面](https://citl.illinois.edu/test-placement-proficiency/current-cutoffs/math-placement) 确认自己学院的 placement 规则。

## 五、重考政策

| 尝试次数 | 要求 |
|----------|------|
| 第 1 次 → 第 2 次 | **24 小时冷却期**（无强制学习时间） |
| 第 2 次 → 第 3 次 | **5 小时 Prep & Learning Module** + 24 小时冷却 |
| 第 3 次 → 第 4 次 | 同上 |
| 第 4 次 → 第 5 次 | 同上 |

- 秋季入学新生：**最多 3 次**评估机会
- 春季入学新生：**最多 2 次**
- 共计 **5 次** 评估 + **12 个月** Learning Module 访问权限
- 取**最高分**作为 Placement 依据
- 成绩有效期：**4 个月**（必须在考试后的下一个学期注册数学课）

## 六、Mock Exam / 模拟题获取渠道

⚠️ **ALEKS PPL 没有官方公开的完整 mock exam**，但有以下几种有效的备考方式：

### 🥇 第一渠道：ALEKS 内置的 Prep & Learning Module（最精准）

| 项目 | 详情 |
|------|------|
| 性质 | 第 1 次考试后自动解锁的**个性化学习模块** |
| 原理 | ALEKS 根据你第 1 次的错误，生成只针对你薄弱环节的练习 |
| 访问方式 | 通过 [UIUC Math ALEKS 页面](https://math.illinois.edu/academics/undergraduate/ALEKS) 登录 |
| 效果 | 官方数据：学习 **6+ 小时**平均提升一个课程等级 |
| 费用 | 免费（含在 placement test 中） |

> 💡 **最佳策略**：尽快完成第 1 次（就当是诊断 mock），然后认真做 ALEKS 推送的 Prep & Learning Module，最后再考第 2、3 次冲 80+。

## 七、考试策略与时间线

### 推荐时间线

```
入学前 5-6 月：收到 UIUC 的 ALEKS PPL 考试通知
         ↓
入学前 6 月初：登录 ALEKS，完成第 1 次考试（当诊断）
         ↓
入学前 6 月-7 月：根据第 1 次结果，做 ALEKS Prep & Learning Module
                  + Khan Academy 补弱项（重点是三角学和指数/对数）
         ↓
入学前 7 月中：完成第 2 次考试
         ↓
（如需）入学前 7 月底-8 月初：再学 Module 5 小时 → 第 3 次考试
         ↓
入学前 8 月（New Student Registration）：用最高分选 MATH 220/221
```

### 冲 80+ 的核心策略

| 策略 | 说明 |
|------|------|
| **不要把第 1 次当正式考** | 第 1 次的真正价值是解锁 Prep & Learning Module。不用紧张。 |
| **认真做 Module** | 系统推送给你的每个 topic 都是你实际不会的，做完就能提分。 |
| **重点攻三角和对数** | 这是区分 70+ 和 80+ 的关键分水岭。 |
| **考试用草稿纸** | 开放作答需要手动计算，务必边算边写。 |
| **检查输入格式** | ALEKS 对数学表达式的输入格式要求严格，考前一定熟悉 ALEKS 的输入界面。 |
| **不要用自己计算器** | 只用 ALEKS 内置的，否则会被判定违规。 |

## 八、常见问题（FAQ）

**Q: 我 AP Calculus BC 拿了 5 分，还要考吗？**
A: 要。UIUC 要求所有新生都必须参加。但 5 分可以使你免修 MATH 220/221，直接进入 MATH 231。

**Q: 考不到 80 分会怎样？**
A: 你会被放在 MATH 115（预备微积分）或 MATH 112。对 CS+GGIS 来说，这意味着你大一秋无法修 MATH 220/221，会导致整个 CS 序列延后一学期。所以**一定要尽力冲 80+**。

**Q: 成绩有效期多久？**
A: 4 个月。必须在考试后的下一个学期注册数学课。

**Q: 考试可以多次考吗？**
A: 可以。秋季入学新生最多 3 次，取最高分。

**Q: 考试在哪里进行？**
A: 在线，自己找安静的环境即可。不需要到校。

**Q: 有中文版吗？**
A: ALEKS 不支持中文界面，但你可以用草稿纸做中文笔记。
