## 1. 核心背景：UIUC CS专业的"封锁令"

### 1.1 政策变化时间线

2022年5月，UIUC官方宣布了一项震动留学圈的决定：**自2022年夏季和秋季学期起，UIUC Grainger工程学院的纯CS专业（BS in Computer Science）永久关闭校内转专业（ICT/IDT）通道**。这意味着：

- ❌ 被录取到UIUC任何非CS专业的学生，**无法**通过校内转专业进入纯CS
- ✅ 新生直接申请CS和外部转学生申请CS通道**不受影响**
- ✅ **CS+X跨学科联合专业**（约16-17个）仍然对校内转专业**开放**

### 1.2 政策背后的原因

- CS申请人数在2015-2020年间增长了约**83%**，但录取名额仅增长约**35%**
- 学院资金和师资无法支撑大量涌入的学生——每位CS学生的教学成本远超学费分配
- 过多转专业学生挤占了正常入读CS学生的选课资源

### 1.3 全国性趋势

UIUC并非孤例。全美Top 50中**越来越多大学正在关闭或严格限制CS内部转专业**：

| 学校 | CS排名 | 内部转CS政策 |
|------|--------|-------------|
| UIUC (纯CS) | #5 | ❌ 2022年起禁止 |
| Georgia Tech | #7 | ❌ 2024年夏季起禁止 |
| Ohio State | #33 | ❌ 2025年5月起禁止 |
| UC Berkeley (EECS) | #2 | ❌ 工程学院EECS禁止转入 |
| Michigan | #10 | 🟠 2023年起严重限制（Advance Selection制度） |
| UCSD | #13 | 🟠 Capped Major，转入率<10% |
| Washington | #7 | 🟠 官方声明"极度不可能" |
| CMU | #3 | 🟠 计算机学院单独审核，成功率个位数 |
| Purdue | #16 | 🟠 名额"极其有限"；2025年起也关闭了外部转CS |
| Maryland | #17 | 🟠 2024年秋季起名额从1000砍至100 |

**核心结论**：全美CS强校的"曲线救国"时代正在终结。UIUC的CS+X是当前少数仍然开放的路径之一。

## 2. 路径A：校内转CS+GGIS

### 2.1 操作可行性 ⭐⭐⭐⭐

#### 什么是CS+GGIS？

CS+GGIS是UIUC的**Siebel计算与数据科学学院**与**地理与地理信息科学系**联合开设的跨学科学位项目，授予**BSLAS（理学学士-文理学院）**学位。UIUC是**CyberGIS（高性能计算应用于地理空间科学）**领域的全美领导者。

#### 学位核心数据

| 指标 | 数据 |
|------|------|
| 总学分 | **120学分** |
| CS核心课程 | ~26-29学分 |
| GGIS专业课程 | ~24-28学分 |
| 数学课程 | ~10-11学分 |
| **平均起薪** | **$108,042** |
| 就业/升学率 | **96%** |
| 4年毕业率 | 81% |
| 平均毕业时间 | 3.7年 |
| 国际生年费用 | ~$72,880 |

#### 课程结构概览

**CS核心课程**：
- CS 124：计算机科学导论 I
- CS 128：计算机科学导论 II
- CS 173：离散数学
- CS 225：数据结构
- CS 222：软件设计实验
- CS 233 + CS 341：计算机体系结构 + 系统编程（**或** CS 340 + 2门CS 400-level选修）
- CS 374：算法与计算模型导论
- CS 421：编程语言与编译器
- CS 361（推荐）：概率与统计

**数学课程**：
- MATH 220/221：微积分 I
- MATH 231：微积分 II
- MATH 225/257：矩阵理论/线性代数

**GGIS核心课程**：
- GGIS 371：空间分析
- GGIS 379：地理信息系统导论
- GGIS 380：地理信息系统 II
- 2门GIS进阶选修（如CyberGIS、遥感、数字制图等）
- 2门人文/自然地理课程

#### 转专业申请流程

1. 确保满足GPA和先修课要求（详见下方）
2. 参加每年**9月和2月**的CS+信息发布会（mandatory）
3. 在信息发布当月的**最后一个周五**前提交申请
4. 等待审核结果

### 2.2 难度 ⭐⭐⭐

#### 硬性门槛

| 条件 | CS+GGIS要求（预估） | 更高要求的CS+X（CS+Math/Stat） |
|------|---------------------|-------------------------------|
| **累计GPA** | ≥ **3.5** | ≥ **3.67** |
| **CS 173（离散数学）** | ≥ **B+** | ≥ **A-** |
| **CS 225（数据结构）** | ≥ **B+** | ≥ **A-** |
| 前置数学课 | 完成微积分I/II | MATH 241/347 需B-以上 |

> ⚠️ **特别注意事项**：
> - 如果通过AP/IB/转学分免修了CS 173和CS 225，则需要在本校修读CS 233和CS 241来替代评估
> - **至少2门CS核心课程必须在UIUC本校完成并取得成绩**（不能全部转学分）
> - 满足条件≠保证录取，CS+X转入仍有竞争

#### 真实难点分析

| 难点 | 具体描述 | 应对策略 |
|------|---------|---------|
| **选课竞争** | 非CS专业选CS 173/225可能满员，优先级低于CS专业学生 | 尽早注册；利用James Scholar优先选课权（如有）；在开放选课时段快速抢课 |
| **课程难度** | CS 225（数据结构）是著名的高难度课程，拿B+以上需要认真投入 | 提前自学C++和基础数据结构；利用office hour和CARE辅导 |
| **GPA维持** | 3.5+意味着每学期至少B+平均水平，不能有短板 | 第一学期课程不宜过重，先稳住GPA再挑战难的课程 |
| **竞争池** | CS+X的转入名额有限，且申请者中不乏高GPA的"曲线救国"者 | 尽早完成先修课，在第一或第二申请周期就提交 |

### 2.3 成功率 ⭐⭐⭐⭐

#### 成功率估算

虽然没有官方公布的CS+X校内转专业录取率，但根据多方信息和社区经验，可以得出以下估算：

- **满足条件的申请者中，CS+X转入成功率估计在50%-70%之间**
- CS+GGIS相对CS+Math/Stat/Physics等"热门X"竞争稍小（GIS方向申请者较少）
- **关键变量**：GPA > 课程成绩 > 申请时机 > 文书/个人陈述

#### 为什么CS+GGIS相对可行？

1. **GGIS专业申请基数小**：与CS+Economics或CS+Statistics相比，GIS方向的竞争池较小
2. **课程重叠度高**：你已录取GGIS，转入CS+GGIS时GGIS部分的课程可以直接沿用
3. **跨学科友好**：GIS本身就是一个需要计算能力支撑的领域，转专业动机天然合理
4. **学校支持CyberGIS方向**：UIUC在该领域投入很大，愿意培养该方向的学生

## 3. 路径B：美本转学读纯CS

### 3.1 操作可行性 ⭐⭐

#### 美本转学的核心挑战

从一所美本（UIUC）转学到另一所美本攻读CS，面临以下关键问题：

**① 4-year university → 4-year university 转学天然处于劣势**

美国大学转学体系**优先录取社区大学（CC）学生**：
- UC系统：**92%的转学名额留给加州社区大学学生**
- UMich：大量名额优先给密歇根社区大学
- UNC：优先北卡社区大学系统
- 作为从另一所4年制大学（UIUC）转出的国际学生，你处于最不占优势的申请池

**② CS专业的特殊壁垒**

即使学校整体转学录取率尚可，**CS专业的实际录取率通常远低于学校平均水平**：
- CMU整体转学录取率5.6%，但SCS（计算机学院）实际录取率或低于2%
- UC Berkeley整体转学录取率25.5%，但EECS专业竞争激烈，国际生实际录取极低
- UCSD转学录取率54.6%，但CSE department每年仅通过selective application录取极少数

**③ 越来越多的学校正在关闭或限制外部CS转学**

- Purdue：**2025年秋季起已关闭外部转学生进入CS的通道**
- UIUC本身：纯CS不接受校内转，外部转CS名额也极为有限
- Maryland：名额从1,000砍至100

**④ 学分的不可控性**

- 转学后学分不一定被目标学校完全认可
- CS专业课尤其严格——课程大纲不匹配可能重修
- 国际生通常需要额外完成WES等成绩认证
- 可能导致毕业延迟1-2个学期

#### 作为国际学生的额外挑战

- 大部分公立大学的转学生助学金/奖学金不向国际生开放
- SEVIS/I-20转移需要两校DSO协调，有行政风险
- 部分学校要求托福≥100（在英语国家学习不足3年），即使已有美本成绩

| CMU SCS | #3 | CS学院单独审核，实际录取率<2% |
| UC Berkeley EECS | #2 | 工程学院EECS几乎不给4-year transfer |

### 3.3 成功率评估 ⭐⭐

#### 现实主义的情景分析

| 情景 | 条件 | 预估成功率 |
|------|------|-----------|
| **最优情景**：GPA 3.9+, 完成了所有目标校CS先修课, 有强科研/竞赛/实习, 申请2-3所Tier 3-4学校 | 大学成绩优异 + 软实力突出 | **30-50%**（至少拿到1个offer） |
| **典型情景**：GPA 3.7-3.8, 完成了大部分先修课, 有些项目经历, 申请5-6所混合梯度 | 符合大多数转学申请者画像 | **15-30%** |
| **保守情景**：GPA 3.5-3.6, 先修课未完全对齐, 软实力一般 | 处于竞争边缘 | **5-15%** |

#### 从UIUC转出的特殊考量

从UIUC（CS排名#5的学校）转出读CS，**你需要在文书中非常审慎地论证"为什么UIUC的CS/CS+X无法满足你的学术需求"**。如果理由不充分（例如仅仅是"想去排名更高的学校"），招生官会认为你的申请缺乏说服力。UIUC的CS资源已经是全美顶尖水平。

**合理的转学理由可能包括**：
- 追求UIUC没有的特定CS细分方向（如某些学校的特色方向）
- 地理位置/行业连接（如硅谷、纽约的实习网络）
- 学校规模/教学模式的偏好差异

## 4. 替代路径与备选方案

### 4.1 UIUC CS Minor（辅修）

如果CS+GGIS转入失败，CS Minor是**最强备选方案**。

#### CS Minor要求

| 课程 | 学分 | 说明 |
|------|------|------|
| CS 124 (CS I) | 3 | 或CS 125 |
| CS 128 (CS II) | 3 | 或CS 126 |
| CS 173 (离散数学) | 3 | 可替代：MATH 213/347 |
| CS 225 (数据结构) | 4 | 必须在本校完成 |
| 2门CS 300/400-level | 6 | 排除CS 397/398/400-403/413/491-499 |
| **最低总学分** | **16+** | 转学分不足需补修 |

**关键优势**：
- CS Minor开放给所有非CS/CE专业的UIUC学生
- 与CS+GGIS的CS核心课程高度重叠，可以并行推进
- 简历上同时显示UIUC学位+CS Minor，求职竞争力不低

### 4.2 其他CS+X方向

如果CS+GGIS竞争意外激烈，可以考虑以下CS+X作为备选：

| CS+X方向 | 预估竞争程度 | 备注 |

| CS+Linguistics | 🟢 低 | NLP方向天然对口 |
| CS+Geography & GIS | 🟡 中 | 你已有GGIS背景 |

| CS+Philosophy | 🟡 中 | 逻辑/AI伦理 |
| CS+Economics | 🔴 高 | 最热门CS+X之一 |
| CS+Statistics | 🔴 高 | 数据科学方向热门 |
| CS+Math | 🔴 高 | 要求GPA 3.67+ |

**策略**：如果CS+GGIS竞争超预期，可以同时考虑CS+Linguistics或CS+Anthropology（竞争相对小，且对NLP/AI伦理方向的职业发展同样有价值）。

### 4.3 纯GGIS + CS Minor路径

即使不转任何专业，**在GGIS专业基础上加CS Minor**，仍然可以： 
- 学习核心CS课程（CS 124 → 128 → 173 → 225 → 374 → 2门upper-level）
- 毕业后从事GIS开发/空间数据科学等技术与地理交叉领域
- 利用UIUC的Career Fair和handshake平台面科技公司
- 申请CS/Data Science方向的硕士项目

### 4.4 硕士转换路径

本科期间保持高GPA + 完成CS Minor课程 → 申请CS Master：
- UIUC本身有Professional MCS（Master of Computer Science）项目
- 可以申请其他学校的CS Master（USC、NEU、NYU等的转码友好项目）
- 这条路径投入产出比很高，且没有"转学失败"的风险

### 🏫 转学选校推荐清单

如果你决定同时走转学路径，推荐的申请组合：

| 档位 | 学校 | 理由 |
|------|------|------|
| **冲刺** (2-3所) | Northwestern, Cornell, UChicago | 私立名校，转学录取率相对高，CS项目强。从UIUC CS+GGIS的角度写转学理由需精心打磨 |
| **主申** (2-3所) | USC, NYU, Michigan | USC/NYU对国际生友好；Michigan CS强但与UIUC差异不大，需评估是否值得 |

### ⚠️ 关键风险提醒

1. **"满足条件 = 确保录取"的陷阱**：CS+X转入是竞争性的，不是达标就录。保持GPA越高越好，同时建立学术导师关系。
2. **"转学录取率高 = 容易"的陷阱**：公立大学的转学录取率包含大量本州CC学生，国际生实际录取率远低于此。
3. **"纯CS > CS+X"的迷思**：在UIUC，CS+X学位在业界的认可度与纯CS几乎没有差异。雇主看的是UIUC CS的课程训练，而非学位名称中的"+"号。
4. **选CS课程的优先级问题**：大一尽快完成CS 124/128，为后续CS 173/225留出时间。如果大一无法选上CS课程，整个转专业时间线会推迟。
