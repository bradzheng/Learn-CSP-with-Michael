**A2.**
$$(-2)^3 \times \frac{3}{4} + \frac{5}{2} \div \left(-\frac{1}{2}\right)$$
$$= (-8) \times \frac{3}{4} + \frac{5}{2} \times (-2)$$
$$= -6 + (-5) = -11$$

> **关键技能:** 运算顺序(PEMDAS), 负数运算, 分数除法=乘倒数

**A3.**
$$\$4.80 \times 0.85 = \$4.08 \quad \text{(八五折)}$$
$$\$4.08 \times 1.10 = \$4.488 \approx \$4.49$$

> **关键技能:** 百分比应用, 连续百分比变化

**A4.**
$$\frac{7}{20} = 0.35 = 35\%$$
$$0.625 = \frac{625}{1000} = \frac{5}{8}$$

> **关键技能:** 分数↔小数↔百分数互化

## Section 2: 代数表达式与线性方程

**A5.**
$$3(2x - 5) - 4(x + 1) + 7$$
$$= 6x - 15 - 4x - 4 + 7$$
$$= 2x - 12$$

> **关键技能:** 分配律, 合并同类项

**A6.**
$$\frac{2x - 1}{3} - \frac{x + 2}{4} = 1$$
两边同乘 12:
$$4(2x - 1) - 3(x + 2) = 12$$
$$8x - 4 - 3x - 6 = 12$$
$$5x - 10 = 12$$
$$5x = 22$$
$$x = \frac{22}{5} = 4.4$$

> **关键技能:** 去分母解线性方程

**A7.**
$$-3 \leq 2x + 1 < 7$$
$$-4 \leq 2x < 6$$
$$-2 \leq x < 3$$
区间表示: $[-2, 3)$

> **关键技能:** 联立不等式, 区间符号（闭区间/开区间）

**A8.**
$$C(m) = 25 + 0.15m$$
当 $C(m) = 70$:
$$70 = 25 + 0.15m$$
$$45 = 0.15m$$
$$m = 300 \text{ miles}$$

> **关键技能:** 建立线性模型, 解释斜率和截距

**A9.**
$$|2x - 5| = 9$$
$$2x - 5 = 9 \quad \text{或} \quad 2x - 5 = -9$$
$$2x = 14 \quad \text{或} \quad 2x = -4$$
$$x = 7 \quad \text{或} \quad x = -2$$

> **关键技能:** 绝对值方程——拆成两个方程

## Section 3: 多项式与因式分解

**A10.**
$$(2x - 3)(x^2 + 4x - 1)$$
$$= 2x(x^2 + 4x - 1) - 3(x^2 + 4x - 1)$$
$$= 2x^3 + 8x^2 - 2x - 3x^2 - 12x + 3$$
$$= 2x^3 + 5x^2 - 14x + 3$$

> **关键技能:** 多项式乘法（分配律逐项展开）

**A11.**
$$3x^3 - 12x$$
$$= 3x(x^2 - 4)$$
$$= 3x(x - 2)(x + 2)$$

> **关键技能:** 提公因式+差平方公式——**彻底分解**到不能分解为止

**A12.**
$$2x^2 + 7x - 15$$
找两个数: 乘积 $= 2 \times (-15) = -30$, 和 $= 7$ → $10$ 和 $-3$
$$= 2x^2 + 10x - 3x - 15$$
$$= 2x(x + 5) - 3(x + 5)$$
$$= (2x - 3)(x + 5)$$

> **关键技能:** $ac$ 分组分解法

**A13.**
$$x^3 - 8 = x^3 - 2^3$$
$$= (x - 2)(x^2 + 2x + 4)$$

> **关键技能:** 立方差公式 $a^3 - b^3 = (a-b)(a^2 + ab + b^2)$

**A14.**
```
        2x^2 +  x + 6
    ┌─────────────────
x-2 │ 2x^3 - 3x^2 + 4x - 5
      2x^3 - 4x^2
      ───────────
             x^2 + 4x
             x^2 - 2x
             ────────
                  6x - 5
                  6x - 12
                  ───────
                      7
```
商: $2x^2 + x + 6$, 余数: $7$

答案: $2x^2 + x + 6 + \frac{7}{x - 2}$

> **关键技能:** 多项式长除法

## Section 4: 二次方程与抛物线

**A15.**
$$2x^2 - 3x - 5 = 0$$
$a = 2, b = -3, c = -5$

判别式: $\Delta = b^2 - 4ac = (-3)^2 - 4(2)(-5) = 9 + 40 = 49$

$$x = \frac{-b \pm \sqrt{\Delta}}{2a} = \frac{3 \pm 7}{4}$$
$$x = \frac{10}{4} = \frac{5}{2} \quad \text{或} \quad x = \frac{-4}{4} = -1$$

> **关键技能:** 求根公式, 正确代入 $a, b, c$

**A16.**
$$x^2 + 6x + 4 = 0$$
$$x^2 + 6x = -4$$
$$x^2 + 6x + 9 = -4 + 9 \quad \text{(两边加}(\frac{6}{2})^2 = 9\text{)}$$
$$(x + 3)^2 = 5$$
$$x + 3 = \pm \sqrt{5}$$
$$x = -3 \pm \sqrt{5}$$

> **关键技能:** 配方法——加一次项系数一半的平方

**A17.**
$f(x) = -x^2 + 4x - 3$, 其中 $a = -1, b = 4, c = -3$

顶点 $x$ 坐标: $h = -\frac{b}{2a} = -\frac{4}{2(-1)} = 2$
顶点 $y$ 坐标: $k = f(2) = -(2)^2 + 4(2) - 3 = -4 + 8 - 3 = 1$

顶点: $(2, 1)$
对称轴: $x = 2$

> **关键技能:** 抛物线顶点公式 $(-\frac{b}{2a}, f(-\frac{b}{2a}))$, $a<0$ 开口向下

**A18.**
$$3x^2 + 2x + 5 = 0$$
判别式: $\Delta = b^2 - 4ac = 2^2 - 4(3)(5) = 4 - 60 = -56$

$\Delta < 0$ → **没有实数解**（两个共轭复根）

> **关键技能:** 判别式 $\Delta$ 判断根的性质: $\Delta > 0$ 两不等实根, $\Delta = 0$ 两相等实根, $\Delta < 0$ 无实根

## Section 5: 有理表达式与分式方程

**A19.**
$$\frac{x^2 - 9}{x^2 + 4x + 3} \cdot \frac{x + 1}{x - 3}$$
$$= \frac{(x-3)(x+3)}{(x+1)(x+3)} \cdot \frac{x+1}{x-3}$$
$$= \frac{\cancel{(x-3)}\cancel{(x+3)}}{\cancel{(x+1)}\cancel{(x+3)}} \cdot \frac{\cancel{x+1}}{\cancel{x-3}} = 1, \quad x \neq 3, -1, -3$$

> **关键技能:** 因式分解后约分, **注明定义域限制**

**A20.**
$$\frac{2}{x - 1} - \frac{3}{x + 2}$$
通分: LCD = $(x - 1)(x + 2)$
$$= \frac{2(x + 2)}{(x - 1)(x + 2)} - \frac{3(x - 1)}{(x - 1)(x + 2)}$$
$$= \frac{2x + 4 - 3x + 3}{(x - 1)(x + 2)} = \frac{-x + 7}{(x - 1)(x + 2)}$$

> **关键技能:** 有理表达式加减——先通分

**A21.**
$$\frac{1}{x} + \frac{1}{x - 2} = \frac{3}{x - 2}$$

注意定义域: $x \neq 0$ 且 $x \neq 2$

两边同乘 $x(x-2)$:
$$(x - 2) + x = 3x$$
$$2x - 2 = 3x$$
$$-2 = x$$
$$x = -2$$

检验: $x = -2$ 不在禁止值中（$x \neq 0, x \neq 2$） ✅

答案: $x = -2$

> **关键技能:** 解分式方程必须检验增根

## Section 6: 根式与有理指数

**A22.**
$$\sqrt{48x^5y^2}$$
$$= \sqrt{16 \cdot 3 \cdot x^4 \cdot x \cdot y^2}$$
$$= \sqrt{16} \cdot \sqrt{x^4} \cdot \sqrt{y^2} \cdot \sqrt{3x}$$
$$= 4x^2|y|\sqrt{3x}$$

由于"假设所有变量为正": $= 4x^2y\sqrt{3x}$

> **关键技能:** 提取完全平方因子

**A23.**
$$\frac{5}{\sqrt{3} - 1}$$
乘以共轭 $\sqrt{3} + 1$:
$$= \frac{5(\sqrt{3} + 1)}{(\sqrt{3} - 1)(\sqrt{3} + 1)} = \frac{5\sqrt{3} + 5}{3 - 1} = \frac{5\sqrt{3} + 5}{2}$$

> **关键技能:** 分母有理化——乘以共轭

**A24.**
$$\left(\frac{8x^{-3}y^6}{27x^6y^{-3}}\right)^{\frac{2}{3}}$$
先化简括号内:
$$= \left(\frac{8y^6y^3}{27x^6x^3}\right)^{\frac{2}{3}} = \left(\frac{8y^9}{27x^9}\right)^{\frac{2}{3}}$$
$$= \frac{8^{\frac{2}{3}} \cdot (y^9)^{\frac{2}{3}}}{27^{\frac{2}{3}} \cdot (x^9)^{\frac{2}{3}}} = \frac{(\sqrt[3]{8})^2 \cdot y^6}{(\sqrt[3]{27})^2 \cdot x^6}$$
$$= \frac{4y^6}{9x^6}$$

> **关键技能:** 负指数移项, 分数指数 = 先开方再乘方

## Section 7: 指数与对数

**A25.**
$$2^{x+1} = 32$$
$$2^{x+1} = 2^5$$
$$x + 1 = 5$$
$$x = 4$$

> **关键技能:** 化为同底指数, 比较指数

**A26.**
$$\log_3\left(\frac{x^2\sqrt{y}}{27z}\right)$$
$$= \log_3(x^2\sqrt{y}) - \log_3(27z)$$
$$= \log_3(x^2) + \log_3(y^{\frac{1}{2}}) - [\log_3(27) + \log_3(z)]$$
$$= 2\log_3(x) + \frac{1}{2}\log_3(y) - 3 - \log_3(z)$$

（因为 $\log_3 27 = \log_3(3^3) = 3$）

> **关键技能:** 对数积商幂法则, 化简对数常数

**A27.**
$$\log_2(x) + \log_2(x - 2) = 3$$
定义域: $x > 0$ 且 $x > 2$, 所以 $x > 2$

合并: $\log_2[x(x - 2)] = 3$
$$x(x - 2) = 2^3 = 8$$
$$x^2 - 2x - 8 = 0$$
$$(x - 4)(x + 2) = 0$$
$$x = 4 \quad \text{或} \quad x = -2$$

检验: $x = -2$ 不满足 $x > 2$ → 舍去
答案: $x = 4$

> **关键技能:** 对数方程化指数, **必须检验定义域**

**A28.**
$$A = 5000(1 + 0.045)^{10}$$
$$= 5000(1.045)^{10}$$
$$= 5000 \times 1.552969...$$
$$\approx \$7,764.85$$

> **关键技能:** 复利公式, 指数计算

## Section 9: 三角函数

**A32.**
$$-\frac{5\pi}{3} \times \frac{180^\circ}{\pi} = -300^\circ$$
加 $360^\circ$: $-300^\circ + 360^\circ = 60^\circ$ → **第一象限** (Quadrant I)

（$-300^\circ$ 等价于逆时针 $60^\circ$）

> **关键技能:** 弧度↔度转换, 共终边角, 判断象限

**A33.**
已知 $\sin\theta = -\frac{3}{5}$, $\theta$ 在 QIV

用 $\sin^2\theta + \cos^2\theta = 1$:
$$\cos^2\theta = 1 - \sin^2\theta = 1 - \frac{9}{25} = \frac{16}{25}$$
QIV 中 $\cos\theta > 0$, 所以 $\cos\theta = \frac{4}{5}$

$$\tan\theta = \frac{\sin\theta}{\cos\theta} = \frac{-3/5}{4/5} = -\frac{3}{4}$$

> **关键技能:** 毕氏恒等式, 各象限三角符号

**A34.**
$240^\circ$ 在第三象限, 参考角 $= 240^\circ - 180^\circ = 60^\circ$
第三象限中 $\sin < 0$:
$$\sin 240^\circ = -\sin 60^\circ = -\frac{\sqrt{3}}{2}$$

> **关键技能:** 参考角法, 特殊角三角值, ASTC法则

**A35.**
$$2\cos^2 x - \cos x - 1 = 0$$
令 $u = \cos x$:
$$2u^2 - u - 1 = 0$$
$$(2u + 1)(u - 1) = 0$$
$$u = -\frac{1}{2} \quad \text{或} \quad u = 1$$

当 $\cos x = 1$: $x = 0$
当 $\cos x = -\frac{1}{2}$: $x = \frac{2\pi}{3}, \frac{4\pi}{3}$

解集: $x \in \left\{0, \frac{2\pi}{3}, \frac{4\pi}{3}\right\}$

> **关键技能:** 三角方程换元法, 单位圆解 $\cos$ 方程

## 🎯 优先级复习路径（按对 GGIS 专业的重要性）

GGIS (Geography & Geographic Information Science) 对数学要求**中等偏高**，建议至少达到 MATH 220 (微积分 I) 水平。

1. **🔴 第一优先级**: Section 2（代数） + Section 4（二次方程）— 所有高阶数学的基础
2. **🟠 第二优先级**: Section 3（因式分解）+ Section 6（根式）— 微积分必备
3. **🟡 第三优先级**: Section 7（指数/对数）— 地理数据建模常用
4. **🟢 第四优先级**: Section 9（三角）— 测量、GIS坐标转换、遥感

## ⏰ 建议复习时间线

| 时间 | 任务 |
|------|------|
| **现在** | 做这套Mock Exam，识别薄弱环节 |

| **第3周** | 再做一遍Mock Exam 中做错的题 |
| **考前1周** | 综合复习，做附加练习题 |
| **考试当天** | 保证充足睡眠，准备纸笔，找安静环境 |

> **重要提示:**
> - ALEKS PPL 是**自适应**考试——你答对越多，题目越难；答错会降低难度
> - 答错了不要慌，后面题目会调整；**认真对待每一题**
> - 第一次考完后，ALEKS 会解锁 Learning Module，这是**最精准的提分工具**——系统知道你哪不会
> - UIUC 允许最多 **5次考试**（Fall: 3次 / Spring: 2次），之间有 24 小时冷却期
> - ⚠️ **一旦考到 80%+，就不能再考了**——所以如果目标是微积分，前几次先熟悉+学习，有信心了再冲刺
