# UIUC James Scholar / LAS Honors 项目完全解读

> 适用对象：UIUC LAS 学院大一新生，希望通过 James Scholar 获得优先选课权以辅助 CS+GGIS 转专业
>
> 更新日期：2025 年 6 月（基于 2025-2026 学年信息）

---

## 一、什么是 James Scholar？

**James Scholar**（全称 Edmund J. James Scholar）是 UIUC **学院级**荣誉项目。每个学院（LAS、工程学院、Gies 商学院等）各自运营，规则略有不同。LAS（文理学院）适用的是 **LAS Honors / LAS James Scholar**。

### 与 Chancellor's Scholar / Campus Honors Program (CHP) 的对比

| | James Scholar (LAS) | Chancellor's Scholar (CHP) |
|---|---|---|
| 层级 | 学院级 | 全校级 |
| 每年新生数量 | 几百人 | ~125 人 |
| 奖学金 | 通常**无** | ~$1,000/年 |
| 选课优先级 | ✅ | ✅ |
| 成绩单标注 | ✅ "Edmund J. James Scholar" | ✅ "Chancellor's Scholar" |
| 小型荣誉研讨课 | 可选 | ✅ 核心特色 |
| 导师制度 | 不强制 | ✅ 强制 |

> **可以同时两个都加入**，CHP 学生会被鼓励同时申请 James Scholar。但对转 CS+GGIS 而言，**James Scholar 就够用了**——核心价值就是选课优先。

---

## 二、核心福利：选课优先级（Priority Registration）

这是对 CS+GGIS 转专业**最关键的福利**。

UIUC 的选课时间（time ticket）按学分+身份排优先级：
- James Scholar：**最早一批**，通常比同年级普通学生早 **1-2 周**
- 普通大一大二学生：学分少，排在最后

CS 173 和 CS 225 是全校最抢手的课，**普通的低年级学生经常抢不到**。James Scholar 的优先选课权意味着你基本能保证选上，这是转 CS+GGIS 的**基础设施保障**。

### 其他福利

- 成绩单上永久标注 "Edmund J. James Scholar"
- 图书馆 stacks 通行权限
- 专门的 Honors Living-Learning Community（可选荣誉宿舍）
- 部分荣誉课程/研讨课可选
- 部分学院提供的 merit award（有限，不保证）

## 三、大一新生如何获取资格？

### 途径 1：录取时自动邀请（最常见）

在提交 UIUC 申请时勾选 **"Honors"** 选项，AO 会根据申请条件自动评估。LAS James Scholar 的自动邀请标准（非官方，根据历年经验）：

| 条件 | 大致门槛 |
|------|----------|
| 高中 GPA | 年级前 **10%-15%**（非硬性，综合评估） |
| ACT | ~**33+**（非强制，因 test-optional 政策） |
| SAT | ~**1500+** |
| 课程 rigor | AP/IB/Honors 数量和成绩 |
| 文书/推荐信 | 综合评估的一部分 |

> ⚠️ **关键提醒**：如果在申请 UIUC 时**没有勾选 honors 选项**，就不会收到自动邀请，但不代表这条路就关了。走途径 2。

### 途径 2：入学后自荐（Self-Nomination）

已入学的 LAS 学生可在满足条件后自己申请：

| 项目 | 详情 |
|------|------|
| **申请窗口** | 每年 **6 月 1 日 – 6 月 30 日** |
| **UIUC GPA 要求** | 官方底线 **3.30**，但自荐竞争通常需要 **3.50+** 更安全 |
| **申请材料** | 在线表格 + **2 篇短文**（你是谁、为什么想加入、目标是什么） |
| **结果公布** | 7 月中旬 |
| **最低承诺** | 自荐生必须在项目中至少待 **2 年 / 4 学期**才能获得荣誉毕业资格 |
| **申请链接** | 通常是 `https://forms.illinois.edu/sec/798326824`（仅在 6 月窗口开放） |

#### 短文建议

两篇短文通常围绕以下主题：
1. **个人介绍** — 你的学术兴趣、背景、为什么对 honours 感兴趣
2. **目标陈述** — LAS Honors 如何帮助你实现学术和职业目标

> 📌 对 CS+GGIS 申请者：可以在短文中提到你对 **计算机科学与地理信息科学交叉领域** 的兴趣，以及优先选课权对你完成 CS 核心课程的重要性。

## 四、维持 James Scholar 身份的要求

### GPA 要求

| 时间点 | 最低 UIUC GPA |
|--------|-------------|
| 第一学期结束时（仅第一年适用） | **3.10** |
| 每学年结束时 | **3.30** |
| 全日制要求 | ≥ 12 学分/学期 |

### 荣誉学分要求

需要积累 "honors points"，通过以下方式获得：
- 修读带 H 标注的荣誉课程（Honors Credit Learning Agreement）
- 参与本科研究
- 完成荣誉论文/项目
- LAS 各年级有具体 honors activity 要求，详见 [LAS Honors Handbook](https://las.illinois.edu/system/files/inline-files/23-24HonorsHandbook.pdf)

> 📌 **关键**：James Scholar 维持 GPA（3.30）**低于** CS+X ICT 门槛（3.50），所以只要你在冲击 ICT，James Scholar 维持条件基本自动满足。

## 五、与其他项目的对比

### UIUC 三层荣誉体系

```
            ┌────────────────────────────┐
            │  Campus Honors / CHP       │  ← 全校级，~125人/年，有奖学金
            │  ┌──────────────────────┐  │
            │  │  James Scholar       │  │  ← 学院级，几百人/年，优先选课
            │  │  ┌────────────────┐  │  │
            │  │  │  Dean's List   │  │  │  ← 学期荣誉，GPA达标自动获得
            │  │  └────────────────┘  │  │
            │  └──────────────────────┘  │
            └────────────────────────────┘
```

- **Dean's List**：学期 GPA 达标（通常 3.80+）自动获得，无需申请，无选课福利
- **James Scholar**：学院级荣誉项目，核心福利是选课优先
- **Chancellor's Scholar (CHP)**：全校级精英项目，小额奖学金 + 小型研讨课 + 导师制

## 六、与 CS+GGIS 转专业策略的联动

### 场景 A：入学时已获 James Scholar 邀请

```
大一秋 → 优先选课，抢到 CS 124 + MATH 220 ✅
大一春 → 优先选课，抢到 CS 128 + CS 173 ✅
大二秋 → 优先选课，抢到 CS 225 ✅
大二秋/春 → CS 173 + CS 225 均 B+，GPA 3.50+，提交 ICT 申请 ✅
```

### 场景 B：入学时没有 James Scholar

```
大一秋：
  - 第一学期 GPA 冲到 3.50+
  - 拼命抢课（靠运气 + 候补名单 + 灵活选 section）
  - 利用 office hour 找教授获得 overrides

大一春 / 大一结束（6 月）：
  - 通过 self-nomination 申请 James Scholar
  - 用大一 3.50+ GPA 作为申请资本
  - 写短文强调 CS+GIS 交叉领域兴趣

大二秋起：
  - James Scholar 身份生效，优先选课
  - CS 225 选课有保障
  - 提交 ICT 申请
```

### 场景 C：始终没有 James Scholar

```
大一大二：
  - 尽可能多修学分，提升 registration time ticket 排名
  - 关注 course notification apps（如 Coursicle 等课程监控工具）
  - 选课时选冷门 section、早上 8 点等竞争小的时段
  - 联系 academic advisor 看是否有课程保留或 overrides
  - 如果实在选不到 CS 173/225，考虑暑期修课（需确认是否被 ICT 认可）
```

## 七、择重点

| 时间 | 行动 |
|------|------|
| **现在** | 确认申请 UIUC 时是否勾选了 honors 选项；如已勾选，查录取通知是否有 James Scholar 邀请 |
| **收到 offer 后** | 如有 James Scholar 邀请，**第一时间接受**，不要错过接受截止日期 |
| **没被邀请** | 不慌。大一秋季全力冲 GPA 3.50+，6 月走 self-nomination |
| **大一秋季** | 即使没有优先选课权，也要尽早注册——关注 registration time ticket 发放日期，第一时间抢课 |
| **备选** | 同时关注 **Campus Honors Program (CHP)** 的校内申请，如果大一 GPA 极高（3.80+），CHP 的选课优先级同样强大 |
