**A15.**
设 $P(x) = (x^4-1)Q(x) + R(x)$, 其中 $\deg R < 4$

$R(x) = ax^3 + bx^2 + cx + d$

已知条件:
- $P(x) \equiv 2x+3 \pmod{x^2+1}$ → $R(x) \equiv 2x+3 \pmod{x^2+1}$
- $P(x) \equiv x-1 \pmod{x^2-1}$ → $R(x) \equiv x-1 \pmod{x^2-1}$

**方法一: 用 $x^2 = -1$ 化简**
在 $\pmod{x^2+1}$ 下，$x^2 = -1$, $x^3 = -x$:
$R(x) = a(-x) + b(-1) + cx + d = (-a + c)x + (-b + d)$
$\equiv 2x + 3$

所以:
$$\begin{cases} -a + c = 2 \\ -b + d = 3 \end{cases} \quad\text{(1)}$$

**方法二: 用 $x^2 = 1$ 化简**
在 $\pmod{x^2-1}$ 下，$x^2 = 1$, $x^3 = x$:
$R(x) = a(x) + b(1) + cx + d = (a + c)x + (b + d)$
$\equiv x - 1$

所以:
$$\begin{cases} a + c = 1 \\ b + d = -1 \end{cases} \quad\text{(2)}$$

由 (1)(2):
$-a+c = 2$ 且 $a+c = 1$ → 相加: $2c = 3$ → $c = \frac{3}{2}$, $a = -\frac{1}{2}$
$-b+d = 3$ 且 $b+d = -1$ → 相加: $2d = 2$ → $d = 1$, $b = -2$

余式: $R(x) = -\frac{1}{2}x^3 - 2x^2 + \frac{3}{2}x + 1$

> **核心技巧:** 利用 $x^2 \equiv -1 \pmod{x^2+1}$ 和 $x^2 \equiv 1 \pmod{x^2-1}$ 降次。

$B = \log_3 4$ → $3^B = 4$

两边 $\log_2: A \leq B$

考虑函数 $f(x) = \frac{\ln(x+1)}{\ln x}$, $x > 0, x \neq 1$

比较 $A = \frac{\ln 3}{\ln 2}$ 和 $B = \frac{\ln 4}{\ln 3}$

$$A - B = \frac{\ln 3}{\ln 2} - \frac{\ln 4}{\ln 3} = \frac{(\ln 3)^2 - \ln 2 \cdot \ln 4}{\ln 2 \cdot \ln 3}$$
$$= \frac{(\ln 3)^2 - \ln 2 \cdot 2\ln 2}{\ln 2 \cdot \ln 3} = \frac{(\ln 3)^2 - 2(\ln 2)^2}{\ln 2 \cdot \ln 3}$$

$\ln 3 \approx 1.0986$, $(\ln 3)^2 \approx 1.2069$
$2(\ln 2)^2 \approx 2 \times 0.48045 = 0.9609$

$1.2069 > 0.9609$ → 分子 > 0 → $A > B$

**证法二（更严格，不用近似值）:**

正确推导:
$\log_2 3 > \log_3 4$
$\iff \frac{\ln 3}{\ln 2} > \frac{\ln 4}{\ln 3}$
$\iff (\ln 3)^2 > \ln 2 \cdot \ln 4 = 2(\ln 2)^2$
$\iff \frac{(\ln 3)^2}{(\ln 2)^2} > 2$
$\iff (\log_2 3)^2 > 2$

所以 $3 > 2^{\sqrt{2}}$, 因此 $\log_2 3 > \sqrt{2}$, 所以 $(\log_2 3)^2 > 2$, 所以 $A > B$

$= (\sin^2 x + \cos^2 x) + 2\sin x\cos x - 1$

所以:
$$= \frac{2\sin x(\sin x + 1)}{\sin 2x} = \frac{2\sin x(\sin x + 1)}{2\sin x \cos x} = \frac{\sin x + 1}{\cos x} = \frac{1 + \sin x}{\cos x}$$

✅ **恒等式成立。**

---

**A43.**
已知: $a = \sqrt{6}$, $b = 2$, $\angle A = 60^\circ$

用正弦定理: $\frac{\sin B}{b} = \frac{\sin A}{a}$

$$\sin B = \frac{b \sin A}{a} = \frac{2 \cdot \frac{\sqrt{3}}{2}}{\sqrt{6}} = \frac{\sqrt{3}}{\sqrt{6}} = \frac{1}{\sqrt{2}} = \frac{\sqrt{2}}{2}$$

$\angle B = 45^\circ$ 或 $\angle B = 135^\circ$（$\sin$ 在一二象限均为正）

检查是否两边都合法（$\angle A + \angle B < 180^\circ$）:
- $60^\circ + 45^\circ = 105^\circ < 180^\circ$ ✅
- $60^\circ + 135^\circ = 195^\circ > 180^\circ$ ❌

**只有一解: $\angle B = 45^\circ$**

这是**SSA 的歧义情况**——虽然 $\sin B$ 有两个可能值，但其中一个使三角形内角和超过 $180^\circ$，故舍去。

**A44.**
$$\cos 3x + \cos x = 0$$

用和差化积公式 $\cos \alpha + \cos \beta = 2\cos\frac{\alpha+\beta}{2}\cos\frac{\alpha-\beta}{2}$:

$$2\cos\frac{4x}{2}\cos\frac{2x}{2} = 0$$
$$2\cos 2x \cos x = 0$$

$\cos 2x = 0$ 或 $\cos x = 0$

**$\cos 2x = 0$:**
$2x = \frac{\pi}{2} + n\pi$
$x = \frac{\pi}{4} + \frac{n\pi}{2}$

在 $[0, 2\pi)$ 中: $x = \frac{\pi}{4}, \frac{3\pi}{4}, \frac{5\pi}{4}, \frac{7\pi}{4}$

**$\cos x = 0$:**
$x = \frac{\pi}{2}, \frac{3\pi}{2}$

解集: $x \in \left\{\frac{\pi}{4}, \frac{\pi}{2}, \frac{3\pi}{4}, \frac{5\pi}{4}, \frac{3\pi}{2}, \frac{7\pi}{4}\right\}$

**A45.**
求 $\sin\left(\frac{1}{2}\cos^{-1}\left(-\frac{3}{5}\right)\right)$

令 $\theta = \cos^{-1}\left(-\frac{3}{5}\right)$。则 $\cos\theta = -\frac{3}{5}$, $\theta \in [0, \pi]$

因为 $\cos\theta < 0$, $\theta$ 在第二象限: $\sin\theta = \sqrt{1 - \frac{9}{25}} = \frac{4}{5} > 0$

用半角公式:
$$\sin\left(\frac{\theta}{2}\right) = \sqrt{\frac{1 - \cos\theta}{2}}$$

因为 $\theta \in (\frac{\pi}{2}, \pi)$, $\frac{\theta}{2} \in (\frac{\pi}{4}, \frac{\pi}{2})$, $\sin\frac{\theta}{2} > 0$, 取正号。

$$\sin\left(\frac{\theta}{2}\right) = \sqrt{\frac{1 - (-\frac{3}{5})}{2}} = \sqrt{\frac{1 + \frac{3}{5}}{2}} = \sqrt{\frac{\frac{8}{5}}{2}} = \sqrt{\frac{4}{5}} = \frac{2}{\sqrt{5}} = \frac{2\sqrt{5}}{5}$$
